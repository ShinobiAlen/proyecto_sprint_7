# My_proyect
# 🚗 Análisis de Vehículos Usados

Aplicación web interactiva desarrollada con Streamlit para explorar datos de vehículos usados.

## 🔗 Demo
https://proyecto-sprint-7-mdxn.onrender.com/

## 📊 Funcionalidades
- Filtros por rango de precio
- Visualización de precio vs kilometraje
- Análisis por condición del vehículo

## 🛠️ Tecnologías
- Python
- Pandas
- Plotly
- Streamlit

## 📌 Objetivo
Identificar patrones en el mercado de vehículos usados para entender factores que influyen en el precio.
